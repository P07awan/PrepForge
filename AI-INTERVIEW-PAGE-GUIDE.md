## âœ… What Happens After You Click "Start"?

1. **Form Validation**
   - Checks if domain is filled
   - Checks if all required fields have values

2. **API Call**
   - Sends POST request to `/api/ai-interviews`
   - Payload:
     ```json
     {
       "interviewType": "TECHNICAL",
       "topic": "Machine Learning",
       "difficulty": "MEDIUM",
       "duration": 30
     }
     ```

3. **AI Question Generation**
   - Backend calls AI service
   - Generates 5 questions based on type/domain/difficulty

4. **Database Storage**
   - Creates AIInterview record
   - Creates Question records
   - Returns interview ID

5. **Navigation**
   - Redirects to: `/interview-room/{interviewId}`
   - Shows generated questions
   - Ready to start interview
